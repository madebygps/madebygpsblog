<p>In Debian-based Linux distributions, the default software manager is the Advanced Packaging Tool, <code>apt</code>.</p><p>Let's take a look at </p><pre><code class="language-bash">apt-cache search
apt-get install
apt-get remove
apt-get purge 
apt-get update
apt-get upgrade
/etc/apt/sources.list</code></pre><h2 id="the-apt-cache-search-command">The <em><code>apt-cache search</code> </em>command</h2><p>Servers that hold the software for particular distributions of Linux are known as repositories. The repositories your system will search in are stored in the sources.list file. To check if a package is available for you to download:</p><pre><code class="language-bash">apt-cache search keyword</code></pre><p>Let's search for the <code>snort</code> keyword</p><pre><code class="language-bash">apt-cache search snort</code></pre><figure class="kg-card kg-image-card"><img src="https://madebygps.azurewebsites.net/content/images/2021/02/DeepinScreenshot_select-area_20210218103012.png" class="kg-image" alt loading="lazy" width="1056" height="527" srcset="https://madebygps.azurewebsites.net/content/images/size/w600/2021/02/DeepinScreenshot_select-area_20210218103012.png 600w, https://madebygps.azurewebsites.net/content/images/size/w1000/2021/02/DeepinScreenshot_select-area_20210218103012.png 1000w, https://madebygps.azurewebsites.net/content/images/2021/02/DeepinScreenshot_select-area_20210218103012.png 1056w" sizes="(min-width: 720px) 720px"></figure><p>Several options were returned, it's up to us to make sure we select the correct one to install.</p><h2 id="the-apt-get-install-command">The <em><code>apt-get install</code></em> command</h2><p>Once we know which package we want to install, we can use</p><pre><code class="language-bash">apt-get install packagename</code></pre><p>to install it, let's install the <code>snort - flexible network intrusion detection</code> system package, from our previous command results. </p><pre><code class="language-bash">apt-get install snort</code></pre><figure class="kg-card kg-image-card"><img src="https://madebygps.azurewebsites.net/content/images/2021/02/Screenshot-from-2021-02-18-10-34-31.png" class="kg-image" alt loading="lazy" width="1059" height="261" srcset="https://madebygps.azurewebsites.net/content/images/size/w600/2021/02/Screenshot-from-2021-02-18-10-34-31.png 600w, https://madebygps.azurewebsites.net/content/images/size/w1000/2021/02/Screenshot-from-2021-02-18-10-34-31.png 1000w, https://madebygps.azurewebsites.net/content/images/2021/02/Screenshot-from-2021-02-18-10-34-31.png 1059w" sizes="(min-width: 720px) 720px"></figure><p>Snort is already installed on my machine so I am not asked Y/n to install it, output might be different for you.</p><h2 id="the-apt-get-remove-command">The <em><code>apt-get remove</code> </em>command</h2><p>If you need to remove a command, use:</p><pre><code class="language-bash">apt-get remove packagename</code></pre><p>The <code>remove</code> command doesn't remove configuration files, for that we use <code>purge</code><em>.</em></p><h2 id="the-apt-get-purge-command">The <em><code>apt-get purge</code></em> command</h2><p>To keep things modular, many Linux packages are broken into software units that many different programs can share. When you install a package, you also install any dependencies or libraries it needs to run, in order to remove all those files, we use the purge command</p><pre><code class="language-bash">apt-get purge packagename</code></pre><p>If I wanted to remove snort and all it's dependencies, I can do</p><pre><code class="language-bash">apt-get purge snort</code></pre><figure class="kg-card Enter Y if you want to continue. </p><h2 id="the-sources-list-file">The <em><code>sources.list</code></em> file</h2><p>This file tells your system which repositories to search for software from. I'll use VI to view it.</p><pre><code class="language-bash">vi /etc/apt/sources.list</code></pre><figure class="kg-card kg-image-card"><img src="https://madebygps.azurewebsites.net/content/images/2021/02/Screenshot-from-2021-02-18-11-07-04.png" class="kg-image" alt loading="lazy" width="1068" height="680" srcset="https://madebygps.azurewebsites.net/content/images/size/w600/2021/02/Screenshot-from-2021-02-18-11-07-04.png 600w, https://madebygps.azurewebsites.net/content/images/size/w1000/2021/02/Screenshot-from-2021-02-18-11-07-04.png 1000w, https://madebygps.azurewebsites.net/content/images/2021/02/Screenshot-from-2021-02-18-11-07-04.png 1068w" sizes="(min-width: 720px) 720px"></figure><p>When you download a package, apt will look sequentially through your sources.list, it stops when it finds the package. </p><p>There are a few categories for repositories</p><ul><li><strong>main: </strong>Contains supported open source software.</li><li><strong>universe: </strong>Contains community-maintained open source software.</li><li><strong>multiverse: </strong>Contains software restricted by copyright or other legal issues.</li><li><strong>restricted: </strong>Contains proprietary device drivers.</li><li><strong>backports: </strong>Contains packages from later releases. </li></ul><p>You can add a repository by editing this file.</p>
