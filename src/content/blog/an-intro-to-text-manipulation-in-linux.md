<p>In Linux, almost everything is a text file, manipulating text is crucial. Let's take look at a few commands that will help us do so:</p><pre><code class="language-bash">cat
head
tail
nl
grep
sed
more
less</code></pre><h2 id="the-cat-command">The <em><code>cat</code> </em>command</h2><p>The most basic command to display text.</p><pre><code class="language-bash">cat file</code></pre><pre><code class="language-bash">cat /usr/share/metasploit-framework/data/wordlists/passwd.lst</code></pre><figure class="kg-card kg-image-card"><img <em>head </em>command</h2><p>This command will display the first 10 lines (by default) of a file. </p><pre><code class="language-bash">head filename</code></pre><p>If you wanted to display a specific number of lines, specify it after the command with a <code>-</code></p><pre><code class="language-bash">head -20 passwd.lst</code></pre><figure class="kg-card kg-image-card"><img opposite of <code>head</code>, will display the last 10 lines (by default) of a file.</p><pre><code class="language-bash">tail filename</code></pre><p>If you wanted to display a specify number of lines, specify it after the command with a <code>-</code></p><pre><code class="language-bash">tail -20 passwd.lst</code></pre><figure class="kg-card kg-image-card"><img file with line numbers.</p><pre><code class="language-bash">nl file</code></pre><p>If you pipe output to it, it makes referencing a whole lot easier.</p><pre><code class="language-bash">head -35 passwd.lst | nl</code></pre><figure class="kg-card kg-image-card"><img src="https://madebygps.azurewebsites.net/content/images/2021/02/image-12.png" class="kg-image" alt loading="lazy" width="1194" </em>command</h2><p>Probably the most widely used text manipulation command. There are entire books on this single command.</p><pre><code class="language-bash">cat file | grep keyword</code></pre><p>It lets you filter content of a file for display.</p><pre><code class="language-bash">cat passwd.list | grep output</code></pre><figure class="kg-card kg-image-card"><img src="https://madebygps.azurewebsites.net/content/images/2021/02/image-13.png" class="kg-image" alt loading="lazy" width="1197" height="174" srcset="https://madebygps.azurewebsites.net/content/images/size/w600/2021/02/image-13.png 600w, https://madebygps.azurewebsites.net/content/images/size/w1000/2021/02/image-13.png 1000w, https://madebygps.azurewebsites.net/content/images/2021/02/image-13.png 1197w" sizes="(min-width: 720px) 720px"></figure><p>Combine it with previous commands, let's view all words from lines 20 to 50 of passwd.lst that have 23 in them</p><pre><code class="language-bash">head -50 passwd.lst | tail -30 | grep 23 | nl</code></pre><ul><li><code>head -50 passwd.lst</code> indicates we want to display the first 50 lines</li><li><code>tail -30</code><strong><em> </em></strong>indicts we want the last 30 lines of those 50 lines from the head command, so lines 20 - 50.</li><li><code>grep 23</code> will display only lines that have 23 in them.</li><li><code>nl</code> will display line numbers.</li></ul><figure class="kg-card kg-image-card"><img src="https://madebygps.azurewebsites.net/content/images/2021/02/image-14.png" class="kg-image" alt loading="lazy" width="1196" height="291" srcset="https://madebygps.azurewebsites.net/content/images/size/w600/2021/02/image-14.png 600w, https://madebygps.azurewebsites.net/content/images/size/w1000/2021/02/image-14.png 1000w, https://madebygps.azurewebsites.net/content/images/2021/02/image-14.png 1196w" sizes="(min-width: 720px) 720px"></figure><h2 id="the-sed-command">The <em><code>sed</code> </em>command</h2><p>This command lets you search for occurrences of a word or a test pattern and then perform some action on it. Similar to Find and Replace in Windows. Commands in <code>sed</code> begin with a single letter.</p><p><code>s</code> is the substitution command</p><pre><code class="language-bash">echo "gwyn" | sed 's/gwyn/gps'</code></pre><figure class="kg-card kg-image-card"><img src="https://madebygps.azurewebsites.net/content/images/2021/02/image-15.png" class="kg-image" alt loading="lazy" width="1920" height="240" srcset="https://madebygps.azurewebsites.net/content/images/size/w600/2021/02/image-15.png 600w, https://madebygps.azurewebsites.net/content/images/size/w1000/2021/02/image-15.png 1000w, https://madebygps.azurewebsites.net/content/images/size/w1600/2021/02/image-15.png 1600w, https://madebygps.azurewebsites.net/content/images/2021/02/image-15.png 1920w" sizes="(min-width: 720px) 720px"></figure><p>Let's use <code>sed</code> to find all the instances of <code>mysl</code> in the <code>snort.conf</code> file and replace it with <code>MySQL</code> and save the file as <code>snorttest.conf</code> in our current directory</p><pre><code class="language-bash">sed s/searchterm/replacementterm/occurence</code></pre><p>The <code>g</code> here stands for global and means replace all instances of the occurrence.</p><pre><code class="language-bash">sed s/mysql/MySQL/g /etc/snort/snort.conf &gt; snorttest.conf</code></pre><figure class="kg-card kg-image-card"><img src="https://madebygps.azurewebsites.net/content/images/2021/02/image-16.png" class="kg-image" alt loading="lazy" width="1920" height="252" srcset="https://madebygps.azurewebsites.net/content/images/size/w600/2021/02/image-16.png 600w, https://madebygps.azurewebsites.net/content/images/size/w1000/2021/02/image-16.png 1000w, https://madebygps.azurewebsites.net/content/images/size/w1600/2021/02/image-16.png 1600w, https://madebygps.azurewebsites.net/content/images/2021/02/image-16.png 1920w" sizes="(min-width: 720px) 720px"></figure><p>You can specify which occurrence of the line you want to replace by using <code>/1</code>, <code>/2</code>, etc, at the end.</p><p>Let's say we have this <code>textfile.txt</code></p><pre><code class="language-text">gwyn gps
